Shadow Escape is a fully playable 2D trap platformer game built in a single HTML file — no downloads or installs needed, just open in any browser.
What it is:
A challenging side-scrolling platformer inspired by troll/trap games like Devil Level, where the goal is simple — reach the exit portal — but the path is filled with hidden dangers designed to kill you repeatedly.
22 hand-crafted levels spread across 6 visual themes: Dark Dungeon, Haunted Forest, Lava World, Cyber City, Castle Keep, and Ice Cavern. Each theme has its own color palette, atmosphere, and trap style. Difficulty scales progressively from a gentle intro all the way to high-speed gauntlets with simultaneous moving platforms, fake floors, and spike fields.
3 playable characters, each with a distinct ability:

🥷 Ninja — fast movement and double jump
🦾 Cyborg — highest jump and hover glide, with a fully animated realistic humanoid robot design
⚔️ Knight — wall jump and slow fall

Core trap types include hidden spikes, fake/disappearing platforms, horizontally and vertically moving platforms, spike rain from above, falling crushers, and lava floors with no ground beneath you.
Full game UI — start menu, character select, level select with a one-by-one unlock system (levels only open after clearing the previous one), pause menu, death counter, game over screen, level clear screen with time and death stats, and an all-clear ending screen.
Tech: Pure vanilla HTML, CSS, and JavaScript — no libraries, no frameworks, no server needed. Pixel-perfect canvas rendering with particle effects, player trail, glowing portals, animated stars, and a subtle grid overlay that changes color per theme
